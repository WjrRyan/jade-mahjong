# Jade Mahjong PRD

## 1. 产品概述

Jade Mahjong 是一款移动优先的麻将连连看 Web 游戏，强调：

- 轻松进入
- 清晰可读
- 低学习成本
- 单局反馈明确
- 可持续解锁与重复游玩

产品不是做成重规则麻将，也不是只做一个静态展示页，而是做成“打开即可玩、每局有目标、每步有反馈”的轻量游戏产品。

## 2. 背景与问题

同类轻量消除或连连看产品里，常见问题包括：

- 页面像网页工具，不像完整游戏
- 关卡偶发无解，影响体验
- 提示、洗牌等系统能力存在，但操作价值弱
- 通关只剩时间和步数，奖励感不足
- 对局过程中缺少“短线决策”与“好操作反馈”

Jade Mahjong 希望解决这些问题。

## 3. 产品目标

### 3.1 业务目标

- 提供一个可公开展示、可直接部署的完整前端游戏项目
- 形成一个具备作品集价值的交付结果
- 为后续扩展为更完整的休闲游戏产品打基础

### 3.2 用户目标

- 在手机上快速开始一局
- 清楚知道能点什么、该做什么
- 失败时能被适度辅助，而不是直接卡死
- 通关时感受到明显成就反馈

## 4. 目标用户

### 核心用户

- 喜欢轻量消除、麻将配对、休闲拼图的玩家
- 偏好碎片化、短时游玩的移动端用户

### 次级用户

- 希望牌面更大、更清晰的用户
- 需要更低阅读负担和更明确点击反馈的用户

## 5. 核心体验原则

- 简单：规则一眼能懂
- 稳定：关卡可解，避免无谓挫败
- 清晰：牌面、状态、可点击区域明确
- 奖励：每次消除、连击、通关都有反馈
- 克制：系统增加策略感，但不增加复杂门槛

## 6. 功能范围

### 6.1 首页大厅

用户能力：

- 开始当前进度
- 查看解锁关卡
- 查看每关最好成绩
- 调整显示与提示相关设置

页面应承载：

- 产品氛围展示
- 关卡入口
- 设置入口
- 当前进度概览

### 6.2 对局页

用户能力：

- 点击自由牌完成匹配
- 使用 `Hint`
- 使用 `Shuffle`
- 使用 `Smart Recovery`
- 使用 `Undo`
- 使用 `Restart`
- 返回大厅

页面应展示：

- 关卡信息
- 时间/分数/连击/剩余牌等 HUD 信息
- 当前宝牌
- 最近一次得分反馈
- 棋盘与工具栏

### 6.3 结果页

页面应展示：

- 当前关卡是否完成
- 分数
- 步数
- 时间
- 星级
- 最佳连击
- 宝牌消除次数

页面应支持：

- 下一关
- 重开当前关
- 返回大厅

## 7. 核心玩法规则

### 7.1 自由牌规则

一张牌满足以下条件时才可点击：

- 上方没有被更高层牌覆盖
- 左右至少一侧开放

### 7.2 匹配规则

- 两张牌都必须是自由牌
- 两张牌必须为相同牌型
- 匹配成功后一起移除

### 7.3 对局状态

- `playing`：当前还有可执行匹配
- `stalled`：仍有剩余牌，但当前无可匹配对
- `won`：所有牌都被移除

## 8. 差异化设计

### 8.1 可解关卡生成

产品要求所有正式关卡都应存在通关路径，不接受“随机看运气”的体验。

### 8.2 双层辅助工具

- `Shuffle`：轻量工具
- `Smart Recovery`：停局恢复工具

### 8.3 动作型 Hint

`Hint` 不只是告诉玩家“哪里有解”，而是直接准备第一步操作。

### 8.4 宝牌机制

每局随机选出 1 个宝牌牌型：

- 消除宝牌获得额外奖励分
- 奖励可与连击倍率叠加

这是当前版本的重要创新，用来提升：

- 对局中的优先级判断
- 局内兴奋点
- 每局差异化体验

## 9. 计分与奖励

### 9.1 基础奖励

- 通关结算看步数、时间、星级
- 局内看即时得分

### 9.2 连击系统

连续成功消除会提升连击数，达到阈值后触发更高倍率。

### 9.3 宝牌奖励

宝牌消除会产生额外奖励，并可与连击倍率叠加。

## 10. 非功能要求

### 10.1 平台

- 以移动端浏览器为优先
- 支持桌面浏览器访问

### 10.2 可读性

- 牌面与状态信息要足够清晰
- 支持 Senior Mode 提升阅读与点击舒适度

### 10.3 可维护性

- 规则集中在引擎层
- UI 展示不承担核心规则判断
- 支持测试和回归验证

## 11. 成功标准

产品达到以下标准时，可视为当前版本完成目标：

- 可从首页开始完整体验一局并通关
- 20 个关卡可持续推进
- 停局时可以通过恢复手段继续游戏
- 玩家能理解并感知宝牌与连击带来的差异
- 结果页能清楚展示一次对局的主要成绩
- 项目可被公开展示并部署
